# HotUnity
基于ILRuntime 的 Unity脚本使用方式 的 热更新脚本。

### 目的：
- 在热更项目中，HotScript 定义为热更脚本，所有继承该脚本的类，都可以以Unity脚本的方式添加、显示、编辑、引用等，这样极大方便了热更项目的脚本使用。
- 因未考虑性能问题，仅建议小型项目使用。

### 已实现功能：
- 热更项目脚本的Editor下编辑。
- 热更脚本可包含一些基础类型。
- 热更脚本可包含任意Component。
- 热更脚本可包含HotScript。
